# Документация Flask (Next.js App Router + MDX)

Структура предназначена для размещения в папке `app/docs` вашего Next.js‑проекта.

## Содержимое

- `app/docs/DocsNav.tsx` — общий компонент навигации по разделам документации.
- `app/docs/page.mdx` — главная страница раздела `/docs`.
- `app/docs/flask-architecture/page.mdx` — теоретическая часть «Архитектура и реализация веб‑приложений на Flask».
- `app/docs/practice-guide/page.mdx` — методические указания по выполнению практической части.

## Использование

1. Скопируйте папку `app` из архива в корень вашего Next.js (App Router) проекта.
2. Убедитесь, что в `next.config.js` включена поддержка MDX (например, через `@next/mdx`).
3. Запустите проект:
   ```bash
   npm run dev
   ```
4. Откройте в браузере:
   - `/docs` — оглавление и навигация;
   - `/docs/flask-architecture` — теоретическая часть;
   - `/docs/practice-guide` — практическая часть.
